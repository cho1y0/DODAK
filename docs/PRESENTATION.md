## 1. 프로젝트 소개

### 1.1 프로젝트 개요

**Dodak (도닥)** 은 AI 기반 감정 일기 & 심리 케어 플랫폼입니다.

"도닥도닥, 당신의 마음을 어루만져 드려요"

바쁜 일상 속 방치되는 감정을 일기를 통해 기록하고, AI가 분석하여 마음 건강을 관리할 수 있는 서비스입니다.

### 1.2 프로젝트 배경 및 목표

| 항목 | 내용 |
|------|------|
| **문제 인식** | 현대인의 증가하는 우울증·불안장애, 높은 심리 상담 문턱 |
| **해결 방안** | 일기를 통한 자기 감정 인식 + AI 감정 분석 + 24시간 챗봇 상담 |
| **핵심 목표** | 누구나 쉽게 감정을 기록하고 AI 도움으로 마음 건강을 관리 |

## 2. 기술 스택

### 2.1 전체 기술 구성

| 구분 | 기술 | 설명 |
|------|------|------|
| **Backend** | Spring Boot 3.5.7 | REST API 서버 |
| **보안** | Spring Security 6 | 인증/인가 |
| **ORM** | Spring Data JPA + QueryDSL 5.0 | 데이터 접근 |
| **AI Server** | FastAPI + Uvicorn | AI 모델 서빙 |
| **감정 분류** | KoBERT | 8가지 감정 + 우울증 위험도 |
| **대화 생성** | KoGPT2 | AI 챗봇 상담 |
| **크롤링** | Selenium | YouTube 음악 검색 |
| **Frontend** | Thymeleaf + JavaScript | SSR 기반 UI |
| **DB** | MySQL 8.0 | 관계형 데이터 저장 |
| **빌드** | Maven, Java 17 | 프로젝트 빌드 |

## 4. 주요 기능

### 4.1 기능 목록

| # | 기능 | 설명 | AI 활용 |
|---|------|------|--------|
| 1 | **감정 일기** | 일상의 감정을 자유롭게 기록, 사진 첨부 | - |
| 2 | **AI 감정 분석** | 일기 기반 8가지 감정 비율 + 우울증 위험도 분석 | KoBERT |
| 3 | **AI 챗봇 상담** | 24시간 심리 상담 챗봇 (Simple/Complex) | KoGPT2 + KoBERT |
| 4 | **음악 추천** | 분석된 감정에 맞는 YouTube 음악 추천 | Selenium |
| 5 | **전문 상담 연결** | 전문 상담사/의사 연결 | - |
| 6 | **감정 통계** | 기간별 감정 변화 추이 차트 | - |

### 4.2 8가지 감정 분석

| 감정 | 영문 |
|------|------|
| 기쁨 | Joy |
| 희망 | Hope |
| 중립 | Neutral |
| 분노 | Anger |
| 슬픔 | Sadness |
| 불안 | Anxiety |
| 피로 | Tired |
| 후회 | Regret |

- 우울증 위험도 자동 감지 (위험 키워드 분석을 통한 조기 발견)

### 4.3 서비스 흐름

```
일기 작성 → AI 분석 → 결과 확인 → 음악 추천
                         ↓
                    챗봇 상담
```

## 5. AI API 명세

| 엔드포인트 | 메서드 | 설명 | 모델 |
|-----------|--------|------|------|
| `/diary` | GET | 일기 감정 분석 (8감정 + 우울증) | KoBERT |
| `/chatbot/g` | GET | Simple 챗봇 (가벼운 대화) | KoGPT2 |
| `/chatbot/b` | GET | Complex 챗봇 (심층 상담) | KoBERT |
| `/youtube/search` | POST | 감정 맞춤 음악 검색 | Selenium |

### 감정 분석 응답 예시

```json
{
  "joy": 25.5,
  "hope": 10.0,
  "neutrality": 15.0,
  "anger": 5.0,
  "sadness": 20.0,
  "anxiety": 10.0,
  "tiredness": 8.5,
  "regret": 6.0,
  "depression": 15.0
}
```
